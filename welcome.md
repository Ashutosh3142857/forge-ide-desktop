# Welcome to Forge IDE 🚀

**Powered by Autonomy AI** — A full cloud IDE that runs natively on your desktop.

## Getting Started

1. **Edit** — Click any file in the Explorer to open it
2. **Run** — Press the ▶ Run button to execute your code  
3. **Build** — Press ⚙ Build to package the app for any platform
4. **Deploy** — Deploy to 11 cloud targets including AWS, GCP, Azure, and more

## Features

- 🖊 Code editor with syntax highlighting
- 📁 File tree with create/rename/delete
- 💻 Interactive terminal
- 🤖 Forge AI panel (Autonomy AI)
- ⚙ One-click builds for Windows, macOS, Linux
- 🚀 11 cloud deploy targets

## Keyboard Shortcuts

| Shortcut | Action |
|----------|--------|
| Ctrl+S   | Save file |
| F5       | Run code |
| Ctrl+Shift+B | Build |

## Build Your App

Click the **⚙ Build** button in the toolbar to package Forge IDE as a native desktop app.
No code signing, no admin rights, no extra tools — just Node.js v18+.
